## [0.1.0] - 2026-03-14

### Added

- Initial release of AgDS Vue — Agriculture Design System (AgDS) as accessible Vue 3 components for the Department of Agriculture, Fisheries and Forestry
- 47 components covering layout, navigation, forms, feedback, and typography
- Full Reka UI integration for accessible primitives (Accordion, Tabs, Dialog, Dropdown Menu, Popover, Combobox)
- Design token system using `--agds-*` CSS custom properties
- Single and range date picker with calendar popover
- Autocomplete with single, multi-select, and async variants
- App layout with collapsible sidebar navigation
- Vitest + @testing-library/vue test suite with axe-core accessibility checks
- TypeScript support with bundled `.d.ts` declarations
- ESM and CommonJS build outputs
